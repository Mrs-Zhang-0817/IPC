# 一些小想法
## 1. 盗墓主题
* **正向：** 解谜游戏
* **反向：** 结合秦始皇陵、活埋工匠的传说的逃生游戏
* **玩法设定：** 玩家扮演被困的工匠，利用陵墓内一万个未解之谜和都市传说作为线索，从充满致命机关的秦始皇陵墓中尝试逃生。

## 2. 类似剧本杀的解密互动游戏
* **题材/背景：** 文化悬疑（后续发散至清明上河图/南宋市井）。
* **玩法设定：** 玩家在场景中搜集线索。当收集到足够线索后，可随时对嫌疑 NPC 发起提问。NPC 的心理状态会根据提问准确性发生动态变化，当心理防线崩溃（数值达到一定值）后会认罪，玩家即获得胜利。可设计成单条主线或多个独立的世界副本。

## 3. 黄土高原小人物视角的历史变迁游戏
* **题材/背景：** 黄土高原、农民、种田、小农经济变化、村庄变迁。时间跨度极长，从三国战乱、五胡乱华、盛唐气象到清军入关，再到近现代的鸦片战争和解放战争。
* **玩法设定：** * **方向一（模拟经营）：** 聚焦于种田与村庄建设，类似《星露谷物语》。
    * **方向二（多线叙事）：** 类似《命运石之门》（石头门）的多时空叙事，用小人物的剪影展现农民在时代变迁下的历史地位。

## 4. 清明上河图系列灵感
围绕《清明上河图》和南宋背景，延伸出了以下四种不同的游戏迭代方向：

* **追溯叙事类：** 每一章节发生一个故事。游戏从现代（或特定时间点）拍卖行出现的一件稀世珍宝切入，往回追溯该宝物在历史上各个阶段所经历的传奇故事。
* **拍卖与市井探索类：** 玩家扮演拍卖师。在拍卖结束后可以在小镇里自由移动和探索，通过与镇民交流获取情报（如大人物动向、皇帝最新的喜好），利用信息差在下一次拍卖中拍出高价。游戏画面和情节与《清明上河图》联动，采用仿古式画风。
* **市井侠客类（结合剧本杀解密）：** 玩家扮演行侠仗义的角色，在清明上河图的场景和 NPC 之间活动，通过线索推理去解决市井生活里的各个市井事件。
* **模拟经营类：** 直接在《清明上河图》的背景里开店，做成古风市井模拟经营类游戏。

## 5. 仙侠题材游戏
* **方向一（动作对战）：** 融入各种中国传统武术/修仙招式的打斗类动作游戏。
* **方向二（策略卡牌）：** 将这些传统招式的对战转变成卡牌对决，核心卖点在于设计极其精美的国风卡面与招式组合。

## 6. 中国象棋规则衍生游戏
* **题材/背景：** 中国象棋、传统棋画、将文化元素自然融入游戏机制。
* **玩法设定：** 世界观设定为所有人都在一个巨大的象棋棋局里，每个人对应一个特定棋子，并必须遵守象棋的移动规则。
    * **基础设定：** 敌对两阵营的“帅/将”不能直接见面。
    * **隐藏趣味机制：** 根据象棋规则设计衍生玩法，例如：己方的一个 NPC 挂在另一个 NPC 身上，就可以触发“蹩马脚”机制，从而抵挡对方“马”棋的跨步攻击。
## 7. 民族传统
* **题材/背景：** 男儿三艺/那达慕等传统活动做成游戏，传播民族文化
