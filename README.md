# 1. 项目概述

## 1.1 游戏名称

暂定名：《水德秦陵》

备选名：

- 《黑水秦陵》
- 《秦陵不死药》
- 《玄水地宫》
- 《秦陵秘宝录》

## 1.2 游戏类型

- 2D 解密推理冒险游戏
- 秘宝搜集游戏
- 历史悬疑探索游戏
- 传统文化题材机关解谜游戏

## 1.3 核心概念

玩家扮演一名研究古籍、机关术和古墓结构的探秘者，通过搜集散落民间的秘宝进入秦始皇陵。在陵墓深处，玩家需要利用盗墓技巧、传统文化知识、五行推理和历史线索破解机关，最终抵达秦王墓室，寻找传说中的“不死药”。

游戏并不把传统文化作为简单背景，而是将其设计成机关语言。玩家需要逐渐理解秦陵的底层规则：

```text
秦以水德，尚黑，水胜火，北方属水，万流归秦。
```

## 1.4 一句话卖点

在一座以“秦以水德”为底层规则构筑的秦始皇陵中，玩家通过盗墓技巧和文化推理破解机关，追寻沉睡于秦王墓室的不死药真相。

## 1.5 关键词

```text
秦始皇陵、解密推理、五行、水德、秘宝、盗墓技巧、机关、奇闻怪志、不死药、历史悬疑
```

## 1.6 项目边界

本项目的核心不是战斗、养成或刷装备，而是：

- 环境观察
- 符号推理
- 道具组合
- 机关破解
- 文化线索解读
- 多结局叙事选择

# 2. 游戏定位

## 2.1 目标体验

本游戏希望给玩家提供以下体验：

1. 进入古墓深处的神秘感与压迫感
2. 通过观察细节发现线索的满足感
3. 用历史文化知识破解机关的顿悟感
4. 工具组合带来的探索代入感
5. 对“不死药”真相逐层反转的悬疑感

## 2.2 核心玩家群体

适合玩家：

- 喜欢解谜、推理、密室逃脱的玩家
- 喜欢中国传统文化、历史悬疑、古墓题材的玩家
- 喜欢剧情驱动、章节制探索的玩家
- 喜欢低操作压力、高脑力参与的玩家

不适合方向：

- 不做高强度动作战斗
- 不做复杂养成数值
- 不做刷装备玩法
- 不以恐怖惊吓作为主要卖点

## 2.3 游戏节奏

整体节奏以“探索—发现—推理—破解—反转”为主。

每个章节建议包含：

```text
1 个核心文化主题
1 个主要机关谜题
2-3 个小型观察谜题
1 个工具组合谜题
1 个剧情推进点
1 个秘宝或关键线索
```

## 2.4 设计卖点

| 卖点 | 说明 |
|---|---|
| 秦陵题材 | 使用秦始皇陵、不死药、水银江河等强辨识度元素 |
| 水德暗线 | 将“秦以水德”设计成贯穿全局的解谜语言 |
| 盗墓技巧 | 通过观纹、听风、探土、拓印等方式增强代入感 |
| 文化推理 | 玩家不是回答历史题，而是用历史逻辑破解机关 |
| 多结局 | 根据玩家对不死药的处理方式形成不同结局 |

# 3. 世界观与故事背景

## 3.1 故事背景

传说秦始皇晚年不断派方士出海寻仙，追求长生不死。史书中记载秦始皇陵以水银为百川江河大海，上具天文，下具地理，机关重重，难以窥见其真貌。

民间另有传言：秦始皇所求的不死药并未遗失，而是被封存在陵墓深处。它并非简单的丹药，而是维持秦陵千年不腐、机关千年运行的核心。

主角沈砚在修复一卷残破帛书时，发现其中记载了“黑水入陵，玄药不死”的隐语。与此同时，他自幼佩戴的一枚黑玉残玦开始在夜间渗出冷水般的墨痕，墨痕与帛书残图重合后，竟拼出一段通往秦陵外藏道的路线。

沈砚进入秦陵并非单纯出于寻宝或求药。他的养父曾是一名研究秦陵文献的学者，十年前因追查“黑水残简”失踪，只留下半枚黑玉残玦和一句遗言：“若黑水再响，莫入陵；若你非入不可，记得不要取药。”主角为了追查养父失踪真相、弄清自己的身世，也为了确认不死药传说是否与黑玉残玦有关，开始搜集开启秦陵的秘宝，并最终进入秦始皇陵。

## 3.2 世界观核心规则

秦陵中的机关并非杂乱无章，而是围绕秦朝的政治、信仰和五行观念建立。

核心规则如下：

```text
周为火德，秦以水德胜之。
水色为黑，方位在北。
秦统一六国，故诸流归一。
陵墓以水银作江河，象征天下归秦。
不死药不是逃离死亡，而是维持秩序不灭。
```

## 3.3 主题表达

游戏表层目标是寻找不死药，深层主题则是：

- 人是否应该追求永生
- 文明如何抵抗遗忘
- 权力是否能通过机关和制度延续
- 真正的不朽是肉体不死，还是历史记忆不灭

## 3.4 氛围基调

整体氛围不是单纯恐怖，而是“庄严、幽暗、神秘、压迫”。

秦陵应被表现为一座仍在运转的古代系统：它沉默、庞大、冷峻，但不是混乱的。玩家越深入，越能感受到它背后的秩序感。

# 4. 主角与主要角色

## 4.1 主角

暂定名：沈砚

身份：

- 民间古籍修复师
- 金石学研究者
- 熟悉古墓机关、堪舆和古文字
- 不是传统意义上的盗墓贼，更偏向研究型探秘者
- 自幼佩戴半枚来历不明的黑玉残玦，残玦遇水会显出类似秦篆的暗纹

人物特点：

- 冷静、谨慎、观察力强
- 对历史有敬畏感
- 不相信长生传说，但愿意追查真相
- 对自己的身世长期存疑
- 面对不死药时会产生动摇

主角进入秦陵的直接动机：

1. 查明养父失踪真相
2. 解开黑玉残玦与黑水残简的联系
3. 确认“不死药”是否真实存在
4. 弄清自己反复梦见秦陵水声与黑色墓门的原因

## 4.2 主角能力定位

主角不是靠武力解决问题，而是靠知识、工具和判断力。

他的能力表现为：

- 能读懂残缺铭文
- 能从壁画顺序中判断机关逻辑
- 能用工具判断墓室结构
- 能辨别秘宝真假
- 能在危险中保持推理能力

## 4.3 主角与秦陵的命运链接

主角与秦陵之间存在一条前期隐约、后期揭晓的神秘联系。

前期表现：

- 主角反复梦见黑水、北方墓门和沉在水下的编钟声
- 黑玉残玦靠近秦制器物时会出现水痕
- 部分秦陵机关并不完全排斥主角，像是在“识别”他的存在
- 无名方士留下的残句中多次出现“持半玦者，当归玄水”的隐语

中期暗示：

- 主角的血滴落在黑水玉玦上时，玉玦短暂补全缺失纹路
- 十二金人残片中有一处铭文提到“守陵者之后，不得取药”
- 养父留下的笔记显示，沈砚并非普通弃婴，而是在一场黑水异象后被带回

终局揭示：

沈砚并不是秦始皇血脉的直接后裔，而是当年“守药方士”一支传承断裂后的遗孤。所谓黑玉残玦，本是秦陵核心机关的“识别符”。守药方士一族世代负责阻止不死药被取走，但到近代传承中断，残玦流落民间。沈砚的养父发现真相后试图封存线索，却因此失踪。

因此，主角进入秦陵并不是偶然寻宝，而是秦陵机关、残玦与守药血脉共同把他引回原点。最终他必须决定：继承守药者的职责，还是打破这条延续千年的命运。

## 4.4 引导角色：残简中的无名方士

角色形式：

- 不直接登场
- 通过残卷、壁画、丹方批注、密室留言出现

功能：

- 提供部分线索
- 制造真假混杂的信息
- 引导玩家逐步接近不死药

可能真相：

无名方士可能是当年参与封存不死药的人。他留下的文字既是警告，也是试炼。

## 4.5 对立存在：秦陵机关意志

秦陵没有传统反派，但机关系统本身可以被塑造成一种“无形对手”。

表现：

- 机关会根据玩家错误操作提升风险
- 部分墓室会重新排列
- 兵俑阵会改变朝向
- 水银江河会因操作错误倒流

它不是怪物，而是秦帝国秩序的残余。

# 5. 核心玩法设计

## 5.1 基础玩法循环

```text
进入区域
→ 观察环境
→ 收集线索
→ 使用工具检查异常
→ 将线索记录到笔记
→ 在推理板组合信息
→ 破解机关
→ 获得秘宝或通行权
→ 推进剧情
```

## 5.2 交互对象类型

| 类型 | 功能 |
|---|---|
| 铭文 | 提供文字线索、文化信息、谜题提示 |
| 壁画 | 提供视觉叙事、顺序提示、象征信息 |
| 器物 | 可观察、鉴定、组合或嵌入机关 |
| 机关盘 | 进行五行、方位、符号排列 |
| 暗格 | 通过工具发现，藏有秘宝或残页 |
| 地砖 | 可触发陷阱或组成路径谜题 |
| 门阵 | 章节通行机关，通常需要多线索破解 |
| 水银渠 | 后期大型环境机关，影响风险值 |

## 5.3 玩家能力

玩家的能力以“盗墓技巧”形式存在。

| 技巧 | 解锁阶段 | 作用 |
|---|---|---|
| 观纹 | 序章 | 观察纹样和隐藏符号 |
| 拓印 | 序章 | 复制铭文用于后续比对 |
| 听风 | 第一章 | 判断墙体空腔、风道、暗门 |
| 探土 | 第一章 | 判断地面结构和机关压力 |
| 辨伪 | 第二章 | 判断秘宝、铭文和器物真假 |
| 牵线 | 第三章 | 远程触发机关，规避陷阱 |
| 封气 | 第四章 | 临时封堵毒气和水银蒸汽 |

## 5.4 信息层级

谜题信息分为三层：

1. 表层信息：玩家一眼可见的图案、颜色、方位
2. 推理信息：需要结合笔记与文化知识得到的含义
3. 机关信息：最终可操作的按钮、顺序、道具、路径

示例：

```text
表层：门上有黑色水纹与赤色火纹
推理：秦以水德，水胜火
机关：按下黑色水纹，避开赤色火纹
```

# 6. 特色机制设计

## 6.1 五行循环机关

### 6.1.1 机制核心

游戏中的大量机关使用五行关系作为底层逻辑。

五行相生：

```text
木 → 火 → 土 → 金 → 水 → 木
```

五行相克：

```text
木克土，土克水，水克火，火克金，金克木
```

但在秦陵中，五行最终往往要指向“水”。玩家需要判断当前机关使用的是相生、相克、逆转还是归一。

### 6.1.2 元素符号表

| 五行 | 方位 | 色彩 | 象征 | 常见机关表现 |
|---|---|---|---|---|
| 木 | 东 | 青 | 神树、生长、根系 | 木柱、树纹、青色机关 |
| 火 | 南 | 赤 | 周火、祭火、朱雀 | 火盆、赤门、朱雀纹 |
| 土 | 中 | 黄 | 封土、社稷、陵丘 | 土台、黄砖、中央盘 |
| 金 | 西 | 白 | 青铜、兵戈、斧钺 | 青铜门、兵器、斧痕 |
| 水 | 北 | 黑 | 秦德、江河、玄色 | 黑门、水纹、水银渠 |

### 6.1.3 谜题变化形式

| 形式 | 说明 |
|---|---|
| 相生顺序 | 按照木火土金水排列 |
| 相克顺序 | 根据克制关系排列 |
| 水德归一 | 所有线索最终指向水 |
| 方位对应 | 根据东南中西北定位机关 |
| 颜色对应 | 青赤黄白黑对应五行 |
| 历史覆盖 | 用秦水克周火解释朝代更替 |

---

## 6.2 盗墓工具组合机制

玩家拥有一套工具。工具不是消耗品，而是解谜方式。

| 工具 | 基础用途 | 高级用途 |
|---|---|---|
| 洛阳铲 | 探测空腔 | 判断暗道深度和方向 |
| 墨线 | 标记直线 | 判断地砖是否错位 |
| 火折子 | 照明 | 验氧、点燃机关引线 |
| 铜镜 | 反光 | 照出暗纹、破解镜像文字 |
| 绳钩 | 拉动远处机关 | 安全测试陷阱 |
| 朱砂 | 标记符号 | 补全残缺铭文、镇压异象 |
| 拓包 | 拓印文字 | 跨场景比对铭文 |
| 蜡封 | 封堵孔洞 | 抑制毒气和水银蒸汽 |

组合示例：

| 组合 | 解谜效果 |
|---|---|
| 铜镜 + 火折子 | 反射光线，照出隐藏壁画 |
| 洛阳铲 + 听风 | 判断空墙后是否存在暗室 |
| 朱砂 + 拓印 | 让模糊铭文显现缺字 |
| 墨线 + 地砖 | 找出发生细微位移的机关砖 |
| 蜡封 + 机关孔 | 暂时封住毒气喷口 |
| 绳钩 + 远处拉环 | 在安全距离触发机关 |

## 6.3 线索笔记与推理板

### 6.3.1 线索笔记

所有重要信息会进入“帛书笔记”。

分类包括：

- 历史记录
- 壁画拓片
- 铭文拓片
- 秘宝信息
- 五行关系
- 方位线索
- 主角推测

### 6.3.2 推理板

关键机关需要玩家把线索放到推理板上组合。

示例：

```text
线索一：秦尚黑
线索二：北方属水
线索三：水胜火
线索四：墓门上的朱雀火纹无法开启
```

推理结果：

```text
南方火纹为误导，北方黑水门才是正确入口。
```

### 6.3.3 设计目的

推理板用于减少纯试错，让玩家清楚地知道自己为什么得出答案。

## 6.4 真假秘宝鉴定机制

### 6.4.1 机制说明

玩家会搜集多件秘宝，但部分秘宝存在伪造、误传或后世修补情况。

玩家需要通过观察和线索判断秘宝真伪。

判断依据：

- 材质是否符合时代
- 铭文是否符合秦制
- 纹饰是否与功能相符
- 出土地点是否合理
- 是否与其他线索矛盾

### 6.4.2 假秘宝的作用

假秘宝不一定完全无用，可以承担以下功能：

- 触发错误机关，增加风险值
- 揭示后世盗墓者的故事
- 作为迷惑信息训练玩家辨伪
- 在特定支线中换取真线索

## 6.5 机关风险系统

### 6.5.1 风险来源

- 解谜步骤错误
- 使用错误工具
- 误触陷阱
- 水银机关泄漏
- 毒气区域停留过久
- 破坏文物或机关结构

### 6.5.2 风险表现

| 风险等级 | 表现 |
|---|---|
| 低 | 轻微呼吸声，画面暗角 |
| 中 | 墓室震动，机关提示减少 |
| 高 | 耳鸣、视野模糊、部分线索描述混乱 |
| 极高 | 影响结局，主角可能被黑水污染 |

### 6.5.3 设计目的

风险系统代替频繁死亡，使游戏保持紧张感，同时不破坏解谜节奏。

# 7. 章节与关卡设计

## 7.1 序章：黑水残简

### 场景

- 古籍修复室
- 地表旧祠
- 残破祭坛

### 剧情目标

主角修复残简，发现秦陵不死药线索，并获得第一件秘宝“黑水玉玦”。

### 玩法目标

- 教学观察
- 教学拓印
- 教学道具使用
- 引入秦以水德

### 核心谜题：黑水门

线索：

- 壁画中周人举火，秦人立于黑水旁
- 残简写有“水胜火，秦尚黑”
- 门上有青、赤、黄、白、黑五色按钮

解法：

```text
选择黑色按钮 → 将黑水玉玦嵌入北方槽位
```

设计目的：

让玩家建立“水德 = 黑色 = 北方”的基础认知。

## 7.2 第一章：文公伐树

### 文化来源

秦文公伐南山大梓树的传说。典故中，大梓树被砍后会遇风雨而自合不断；后来有人夜闻鬼语，得知若使人披发，并以朱丝绕树再伐，便可破树神之力。树断后有青牛出，逃入丰水。该传说可改编为神木、朱丝、风雨、青牛、丰水与秦国受命兴起的故事。

### 场景

地下神木祠。

### 场景元素

- 树根穿透墓砖
- 青铜斧痕
- 腐朽祭台
- 缠绕神木的朱丝残痕
- 南壁火纹与被烟熏黑的祭灶
- 青牛浮雕半没于黑水渠口
- 黑水从土缝中渗出
- 供奉神木的石龛

### 核心谜题：神木斧痕

玩家通过壁画得知：

```text
南山有大梓，斧伐而风雨作，树创复合。
病者夜闻鬼语：被发，以朱丝绕树，可困树神。
朱丝属火，火灼其灵；金斧属金，金乃断木。
木倒入土，土裂见青牛，青牛走丰水，黑水由此入秦。
```

此处的关键不是单纯“金克木”。若玩家只用金斧砍树，机关会模拟典故中的“树生合不断”，斧痕会自动愈合。玩家必须先找到朱丝，将其绕在神木机关上，以“火”破除树神自合之力，再用青铜斧触发“金克木”。

正确顺序：

```text
火（朱丝绕树） → 金（青铜斧伐木） → 木（神木断裂） → 土（树根入土） → 水（青牛入丰水，黑水涌出）
```

其中“火”不是最终答案，而是破解神木妖力的前置条件。它对应典故中的朱丝、南山与祭火，也让本谜题完整覆盖五行循环。

### 章节秘宝

断木神斧。

### 章节目的

首次正式把奇闻怪志、五行相克和机关解谜结合起来。

```text
持玦归陵者，非取药人，乃守药人。
```

## 8.1 谜题一：黑水门

### 设计目的

---

## 8.2 谜题二：神木斧痕

### 场景

地下神木祠。

### 线索

- 巨木壁画
- 青铜斧痕指向西方
- 西方属金
- 神木上残留朱丝纹路
- 南壁火纹与祭灶提示“朱为火色、南方属火”
- 壁画中树身被砍后又重新合拢
- 青牛浮雕朝向黑水渠
- 树根下有湿土与黑水

### 解法

按下机关顺序：

```text
火 → 金 → 木 → 土 → 水
```

具体操作为：

1. 在南壁祭灶取得朱丝，绕于神木机关
2. 点亮火纹，使朱丝显现，破除神木自合
3. 使用青铜斧对应西方金位，砍断神木
4. 调整树根机关，使断木落入中央土位
5. 将青牛浮雕推入黑水渠，开启水德暗门

### 设计目的

通过秦文公砍树故事，将朱丝属火、青铜斧属金、神木属木、树根入土、青牛入水串成完整五行链条。玩家会意识到五行并非固定顺序，而会因典故语境产生“前置破法”和“最终归水”的变化。

# 11. 风险、失败与结局系统

## 11.1 失败方式

游戏不以频繁死亡为主，失败更多体现为：

- 风险值上升
- 墓室状态恶化
- 部分线索损坏
- 结局条件变化
- 最终选择受限

## 11.2 风险值影响

风险值会影响：

- 画面效果
- 音效氛围
- 主角笔记内容
- 最终结局描述
- 隐藏结局是否可达成

## 11.3 结局一：取药

玩家拿走不死药。

结果：

- 主角逃出秦陵
- 若风险值过高，暗示主角被黑水污染
- 结尾出现黑水从袖口渗出

主题：对永生的贪欲延续了秦皇执念。

## 11.4 结局二：封药

玩家重新封印不死药。

条件：

- 推理正确率较高
- 没有大量破坏机关
- 识破丹室诱饵

结果：

- 秦陵重新沉入地下
- 主角带走拓片，准备公开历史真相

主题：真正的不朽是文明记忆。

## 11.5 结局三：毁药

玩家破坏不死药核心。

条件：

- 找到足够多的十二金人残片
- 破解最终五行逆转机关

结果：

- 秦陵部分崩塌
- 主角逃生
- 大量历史遗存被毁

主题：破除执念也可能带来代价。

## 11.6 隐藏结局：黑水不息

条件：

- 收集所有真品秘宝
- 完成所有文化支线
- 风险值保持较低
- 最终选择不取、不封、不毁

结果：

- 主角发现不死药不是药物，而是秦陵核心系统
- 秦始皇追求的不是个人长生，而是帝国秩序永不坍塌
- 黑玉残玦补全后，主角确认自己是守药方士传承断裂后的遗孤
- 养父当年并非死于寻宝，而是为了阻止秦陵过早召回主角而被困于外藏道
- 主角没有取药，也没有毁药，而是将黑玉残玦留在秦王墓室，使秦陵不再继续召唤后人
- 主角离开后，陵墓深处仍传来水声，但这一次水声逐渐远去

主题：历史不是死亡，而是循环。

# 14. UI 与交互设计方向

## 14.1 UI 风格

UI 应与古籍、帛书、青铜器纹样结合。

建议元素：

- 帛书笔记
- 竹简菜单
- 青铜纹边框
- 篆文符号按钮
- 水纹加载效果
- 墨迹式转场

## 14.2 主要界面

| 界面 | 功能 |
|---|---|
| 主界面 | 开始游戏、继续、章节选择 |
| 帛书笔记 | 查看线索、历史解释、拓片 |
| 推理板 | 组合线索，得出结论 |
| 道具栏 | 使用工具和秘宝 |
| 秘宝鉴定 | 查看材质、铭文、真伪判断 |
| 风险状态 | 以非数字方式表现危险程度 |

## 14.3 交互原则

1. 重要机关需要明确反馈
2. 错误操作不应立即惩罚过重
3. 推理结果要能回看
4. 文化解释应简洁，不打断探索节奏
5. 鼠标或触控点击区域应清晰

# 15. 叙事节奏与文本风格

## 15.1 叙事方式

- 少量主角独白
- 壁画与铭文叙事
- 残简、手札、丹方提供背景
- 场景变化暗示剧情推进

## 15.2 文本风格

文本应保持古意，但不能过度晦涩。

示例风格：

```text
帛书残字已被水痕浸开，只余一句尚能辨认：
“周火既熄，玄水当兴。”
```

主角推理文本示例：

```text
黑为水色，北为水位。若此门依秦制而建，南侧朱雀火纹多半是诱导。
```

## 15.3 信息呈现原则

- 历史知识先通过场景呈现，再通过笔记解释
- 谜题提示应伪装成考古记录或主角思考
- 不使用过长说明阻断玩家操作

# 16. 最小可行版本规划

## 16.1 试玩版范围

建议先制作一个小型试玩版验证核心玩法。

试玩版内容：

1. 序章：黑水残简
2. 第一章：文公伐树
3. 一个五行机关
4. 一个盗墓工具组合谜题
5. 一个推理板谜题
6. 一个秘宝鉴定环节
7. 一个章节结尾反转

## 16.2 试玩版目标

验证以下问题：

1. 玩家是否能理解“秦以水德”的解谜规则
2. 五行机关是否足够有趣且不晦涩
3. 盗墓工具是否比普通点击更有代入感
4. 文化典故是否自然融入谜题
5. 风险系统是否能增加紧张感但不打断推理

## 16.3 试玩版结束点

试玩版建议结束在第一章末尾：

玩家破解秦文公砍树机关后，获得“断木神斧”。神木根系裂开，露出通往秦陵深处的黑水暗道。主角低声说：

```text
原来这座陵，不是为了埋葬一个帝王。
它是在让整个秦国继续活着。
```

# 18. 设计原则总结

本项目最重要的设计原则是：

```text
用秦文化作为机关语言。
```

玩家不应靠乱点或背答案推进，而应通过观察和推理逐渐读懂秦陵。

核心规则可以总结为：

```text
黑为尊，水为德，北为正，六国归一，万流归秦。
```

当玩家最终抵达秦王墓室时，理想体验不是“终于找到了钥匙”，而是：

```text
我已经读懂了这座陵墓。
```

表层目标是寻找不死药，真正的体验目标是让玩家理解：

```text
所谓不死，也许不是肉体不灭，而是文明、权力、记忆与秩序在历史深处持续回响。
```
